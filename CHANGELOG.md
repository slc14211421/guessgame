## [1.0.0] — 2026-07-10

### 首次发布 🎉

**你比我猜** v1.0 正式发布。一款面向 Android 手机端的离线猜词聚会游戏。

#### 核心功能

- **游戏流程：** 选择分组 → 选择时长 → 5 秒倒计时 → 横屏猜词 → 时间结束提示
- **分类卡片：** 首页横向无限循环轮播选择词语分组
- **猜词时长：** 支持 1/2/3/5/10 分钟可选
- **词条滚动：** 跑马灯滚动大字号展示当前词条，支持切换为静态居中
- **切词交互：** 触摸滑动切词（左右交替入场）、音量键切词（不显示系统音量条）
- **蓝牙遥控：** 支持通用自拍杆蓝牙遥控器切词
- **横屏自适应：** 根据手机横握方向自动切换正反横屏
- **全屏沉浸：** App 端隐藏系统导航栏和状态栏，背景色满屏
- **屏幕常亮：** 游戏进行中防止自动熄屏

#### 词库管理

- 5 个默认分类：成语（500 词）、水果、蔬菜、动物、人物（500 词）
- 分类新增/删除、词语新增/删除
- 删除分类时级联删除该分类下所有词语
- 输入校验：中英文混排宽度计算、空文本拦截、同分类去重
- 本地持久化：分类、词语、设置杀进程重启后保留

#### 用户设置

- 游戏背景色（8 款预设 + 自定义 hex）
- 词条文字颜色（8 款预设 + 自定义 hex）
- 出词方式：顺序 / 随机（Fisher-Yates 洗牌）
- 词条展示：滚动（跑马灯）/ 静态居中
- 设置即时生效并持久保存

#### 音效系统

- 准备开始音效 → 5 秒倒数滴答 → 最后 10 秒提示音 → 游戏结束音效
- Android App 端原生音频播放器优先
- H5 端多路径降级兼容
- iOS 静音模式预热解锁

#### 技术栈

| 模块 | 技术 |
|------|------|
| 应用框架 | uni-app 3.0 |
| 前端框架 | Vue 3.4 + Composition API |
| 开发语言 | TypeScript 5.5 |
| 样式 | SCSS |
| 本地存储 | uni.storage 同步封装 |
| 单元测试 | Vitest 2.1 |

## 版本号规则

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)：

- **主版本号（MAJOR）：** 不兼容的 API 修改或重大功能重构
- **次版本号（MINOR）：** 向下兼容的功能新增
- **修订号（PATCH）：** 向下兼容的问题修正、文档更新

## 发布流程

1. 确保 `npm run typecheck` 和 `npm test` 全部通过
2. 更新 `package.json` 中的 `version` 字段
3. 更新本 `CHANGELOG.md`
4. 提交并打 tag：
   ```bash
   git add -A && git commit -m "release: vX.Y.Z"
   git tag -a vX.Y.Z -m "vX.Y.Z"
   git push origin master --tags
   ```
5. 在 GitHub Releases 页面基于 tag 创建 Release，附上对应版本的 Changelog 内容
