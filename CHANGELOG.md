# Changelog

本文件记录「你比我猜」项目的所有重要版本变更。

---

## [1.0.0] — 2026-07-14

### 首次发布 🎉

**你比我猜** v1.0.0 正式发布。一款面向 Android 手机端的离线猜词聚会游戏。

---

### 游戏流程

- 首页横向无限循环轮播选择词语分组
- 单选猜词时长：1 / 2 / 3 / 5 / 10 分钟
- 点击开始 → 5 秒倒数读秒（含音效）→ 自动进入横屏猜词
- 游戏最后 10 秒倒计时提示音 → 时间到弹出结束提示

### 猜词模式

- 横屏全屏沉浸展示，隐藏系统导航栏和状态栏
- 根据手机横握方向自动切换正反横屏
- 背景色满屏铺满，词条超大字号高对比展示
- 大字号跑马灯滚动（可切换为静态居中）
- 触摸滑动切词，左右交替入场
- 音量键 / 通用自拍杆蓝牙遥控器切词（不弹出系统音量条）
- 游戏进行中屏幕常亮
- 顶部居中加粗展示本局剩余时间（MM:SS 格式）
- 退出后恢复竖屏和系统导航栏

### 词库管理

- 5 个内置分类：成语（500 词）、水果、蔬菜、动物、人物（500 词）
- 分类新增 / 删除，词语新增 / 删除
- 删除分类时级联删除该分类下全部词语
- 中英文混排显示宽度校验（中文 2 单位 / 英文 1 单位，上限 24）
- 同分类下词语自动去重
- 分类、词语、设置本地持久化，重启保留
- 存储版本迁移：升级时同步最新默认词库，保留用户自建词条

### 用户设置

- 游戏背景色：8 款预设 + 自定义 hex
- 词条文字颜色：8 款预设 + 自定义 hex
- 出词方式：顺序 / 随机（Fisher-Yates 洗牌）
- 词条展示：滚动（跑马灯）/ 静态居中
- 颜色可读性自动修正：基于 WCAG 相对亮度标准，对比度不足时自动调整为可读色
- 设置即时生效并持久保存

### 音效系统

- 准备开始 → 5 秒倒数滴答 → 最后 10 秒提示音 → 游戏结束
- Android App 端优先原生音频播放器，H5 端多路径降级
- iOS 静音模式预热解锁

## 版本号规则

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)：

- **主版本号（MAJOR）：** 不兼容的 API 修改或重大功能重构
- **次版本号（MINOR）：** 向下兼容的功能新增
- **修订号（PATCH）：** 向下兼容的问题修正、文档更新

## 发布流程

1. 确保 `npm run typecheck` 和 `npm test` 全部通过
2. 更新 `package.json` 中的 `version` 字段
3. 更新本 `CHANGELOG.md`
4. 提交并打 tag：
   ```bash
   git add -A && git commit -m "release: vX.Y.Z"
   git tag -a vX.Y.Z -m "vX.Y.Z"
   git push origin master --tags
   ```
5. 在 GitHub Releases 页面基于 tag 创建 Release，附上对应版本的 Changelog 内容
